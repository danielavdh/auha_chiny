Ready, Steady, Go!

  1. We are learning from 2 main sources:
    "Invent Your Own Computer Games with Python" by Al Sweigart (download of pdf book free) and
    "Learn Python the Hard Way" by Zed A. Shaw (online book free)
  2. Both books have detailed installation and setup instructions for Python on OSX, Windows and Linux. We have Python installed via pyenv, so it is easy to run different versions of Python. I agree with Zed about giving Idle a miss. We run Python from the Terminal application. For iOS, the Pythonista APP is great. As to Python versions, since this is for a child (= future), we decided for version 3 instead of 2.
  3. Ready to go, when:
    1. $ python  in Terminal starts up an instance of interactive Python. (stop it by pressing "ctrl" and "d")
    2. $ python path_to_filename.py  runs a Python programme saved in filename.py
  4. Disclaimer: If anybody else but ourselves is using any of our resources, do bear in mind that most of the content in these pages is written for the benefit of a young child. I know e.g. that there is a distinction between 'parameters' and 'arguments' of a function. At this stage though, I am happy, if the child I am teaching learns these 2 words and connects them to the stuff in a bracket after the name of a function.
  5. Aim: Learn Python basics and learn how to find information independently to move on. Stack Overflow is a wonderful place to find answers.
