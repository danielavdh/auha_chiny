  • Clipping vs Masking clipping: image goes into e.g. rectangle shape (horizontal blue line)
    masking: image has e.g. rectangle shape (vertical blue line) as a child layer, i.e. vector mask
    clipping allows more flexibility, rectangle like a window, image can be moved around behind. Lock children to manipulate mask with children in set position

    use effects panel to add effects

Retouching

  • Cloning clone on new pixel layer for non-destructive cloning, set to current layer and below or layers beneath
    clone can also be scaled!
  • Inpainting sits under healing brush, brilliant for content replacement
    works best on unprocessed image, so do first!
    duplicate original image into new layer for non-destructive approach
  • Frequency Separation distributes frequencies to separate layers (filters-frequency separation): high frequency (fine detail) and low frequency (tonal information)
    increase radius to filter more detail into high frequeny layer, if needed
    edit the low frequency layer (e.g. remove wrinkles and blemishes)
    then heal the high frequency layer detail with brush set on high hardness
  • Liquify: Retouching dilate pupils (punch tool with pinch setting), make flowers bigger
    use freeze tool to protect areas from being affected by liquifying
    punch tool to make areas larger, twirl to add e.g. smile
    move left or right to shift parts (e.g. slim down arm)
  • Vector-Based Retouching pen tool to enhance lines
    fill, opacity down, add gaussian blur to replace window panes
    group layers that need same processing applied
    gaussian blur, clarity filter with blend mode
