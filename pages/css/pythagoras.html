The mathematical formula for this is
blue area = a x a purple area = b x b green area = c x c
blue area + purple area = green area
a x a + b x b = c x c
or as you will learn in secondary school:
a2 + b2 = c2

That's why we need Mr. Pythagoras!
If you cut the equilateral triangle in half, you get 2 rightangled triangle. They are exactly the same, mirroring each other.
In each of the rightangled triangles, the shortest side is exactly half of the whole side. (You cut it in half, remember?)
This means, we know the long side, we know the short side and thanks to Pythagoras, we can now work out the middlest side. Which is the thickness of the border, or the height of the black arrow.

Shortest side square + middlest side square = longest side square
OR: Longest side square - shortest side square = middlest side square
15 x 15 = 225
7.5 x 7.5 = 56 (rounded that is...)
225 - 56 = 169
Middlest square area = 169
Middlest side = 13
