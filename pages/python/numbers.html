Numbers Exercises:


+  adds numbers

-  subtracts numbers

*  multiplies numbers

/  divides numbers

Type into Terminal: (you type) (python answers)

$ python

>>> 4+3 ↲

7

>>> 4+3+43+77+906+1234-88-99 ↲

2080

>>> 4*5 ↲

20

>>> myAge = 7 ↲

>>> print(myAge) ↲

7

>>> sisterAge = 3 ↲

>>> brotherAge = 12 ↲

>>> childrenAge = myAge + sisterAge + brotherAge ↲

>>> print(childrenAge) ↲

22

# Mixing strings and numbers:

>>> print("The children's age is " + childrenAge + ".") ↲

TypeError: Can't convert 'int' object to str implicitly


# Make the number into a string: str(childrenAge)

>>> print("The children's age is " + str(childrenAge) + ".") ↲

The children's age is 22.


# Or with format strings:

>>> print("The children's age is %d." % childrenAge) ↲

The children's age is 22.

# Adding numbers:

>>> print(3+2+2) ↲

7


# Concatenating strings:

>>> print("3"+"2"+"2") ↲

322


# Both use the + operator!!! But + does different things with strings (lining them up) than with numbers (adding)!
