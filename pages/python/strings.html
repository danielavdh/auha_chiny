String Exercises:

Type into Terminal: (you type) (python answers)

$ python

>>> print("Hello World") ↲

Hello World

>>> print('Strings can also have single quotes around them') ↲

Strings can also have single quotes around them

>>> myName = "Hermione" ↲

>>> print(myName) ↲

Hermione

>>> print("My name is " + myName + ".") ↲

My name is Hermione.


# With a format string you can get the same result, without all the + signs!

>>> print("My name is %s." % myName) ↲

My name is Hermione.

>>> print("I said my name is " + myName + ", " + myName + ", " + myName + ", " + myName + "!") ↲

I said my name is Hermione, Hermione, Hermione, Hermione!


# Instead of writing 3 times the same (myName + ", "), you can times this expression by 3

>>> print("I said my name is " + 3*(myName + ", ") + myName + "!") ↲

I said my name is Hermione, Hermione, Hermione, Hermione!


# Or with the format string trick:

>>> print("My name is %s, %s, %s, %s!" % (myName, myName, myName, myName)) ↲

I said my name is Hermione, Hermione, Hermione, Hermione!

>>> myAge = 7 ↲

>>> print("My name is " + myName + " and I am " + myAge + " years old.") ↲

TypeError: Can't convert 'int' object to str implicitly


# Oops! What happened? 7 is a number, not a string!

# You can make it into a string like so: str(myAge)

>>> print("My name is " + myName + " and I am " + str(myAge) + " years old.") ↲

My name is Hermione and I am 7 years old.


# Or you can use the format string method: (the %s is for string, the %d for digit=Number)

>>> print("My name is %s and I am %d years old." % (myName, myAge)) ↲

My name is Hermione and I am 7 years old.
